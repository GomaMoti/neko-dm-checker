# 🐱 neko-dm-checker-DM怪しいチェッカー

A browser extension for X (Twitter) that uses Grok AI to detect scam patterns in DMs and profiles.<br>
**Japanese-language only. Built with AI assistance.**

---

## 概要

XのDMや怪しいアカウントをGrok AIで分析する、Chrome拡張機能にゃ。<br>
詐欺・不審なDMからユーザーを守ることを目的としているにゃ。

- 公開アカウントはワンクリックで自動分析
- 鍵アカウントはツイートをコピペして分析
- 信頼リスト（信頼/要注意/危険）を自分で管理
- プロンプトテンプレートを自由にカスタマイズ可能

---

## 使い方

1. このリポジトリをダウンロード（ZIP）
2. Chromeで `chrome://extensions/` を開く
3. 「デベロッパーモード」をオンにする
4. 「パッケージ化されていない拡張機能を読み込む」でフォルダを選択する
5. XのDM画面（チャット）を開きDM相手を選択すると右に🐱アイコンが表示される

---

## 機能

### 🔍 アカウント・DM分析
- 公開アカウントはワンクリックでGrok自動分析
- 鍵アカウントはツイートをコピペして分析
- ハンドルをインライン編集して任意のアカウントを調査

### 🚩 チェックリスト（20項目）
- 投資・副業・ロマンス・海外案件・グルーミング・なりすましなど詐欺パターン網羅
- 若年層向け追加チェック項目（オーディション名目・秘密の撮影など）
- カテゴリ別ON/OFFで自分の用途に絞って表示

### ⚖️ フィルター強度
- 弱・普通・強の3段階でGrokの判定指示を調整

### 📋 信頼リスト
- アカウントごとに「信頼／要注意／危険」を登録・管理
- 登録情報はGrokクエリに自動反映

### ⚙️ プロンプトカスタマイズ
- 詐欺チェックの冒頭・末尾・ツイート分析プロンプトを自由に編集
- デフォルトへのリセット機能付き

### 🧒 キッズモード（ファミリー版のみ）
- 小学校高学年〜中学生向けのやさしい判定文に切り替え
- 子供向けチェックリストに自動切り替え
- 見守りガイド・安全ヒント表示

### 🔒 プライバシー
- 入力内容は外部サーバーに送信されません
- データはブラウザローカルに保存

---

## 注意事項

- **デベロッパーモード専用**です（Chrome Web Store未公開）
- 判定結果はあくまで参考です。最終判断はご自身でお願いします
- DM相手には判定結果は見えません（秘密にゃ🐾）
- Xアカウントが必要です
- テストしたところGrokは現在活発じゃないアカウントはだいたい怪しいと騒ぎだします
- 精度よりAI（他の人）に繋げることが重要だと思って自由に騒がせています
