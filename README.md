# 🐱 neko-dm-checker-DM怪しいチェッカー

A browser extension for X (Twitter) that uses Grok AI to detect scam patterns in DMs and profiles.<br>
**Japanese-language only. Built with AI assistance.**

---

## 概要

XのDMや怪しいアカウントをGrok AIで分析する、Chrome拡張機能にゃ。<br>
詐欺・不審なDMからユーザーを守ることを目的としているにゃ。

- 公開アカウントはワンクリックで自動分析
- 鍵アカウントはツイートをコピペして分析
- 信頼リスト（信頼/要注意/危険）を自分で管理
- プロンプトテンプレートを自由にカスタマイズ可能

---

## 使い方

1. このリポジトリをダウンロード（ZIP）
2. Chromeで `chrome://extensions/` を開く
3. 「デベロッパーモード」をオンにする
4. 「パッケージ化されていない拡張機能を読み込む」でフォルダを選択する
5. XのDM画面（チャット）を開きDM相手を選択すると右に🐱アイコンが表示される

---
### 📦 2つのバージョン
好きなスタイルを導入できるように２バージョンあります
1. **🧑‍💻 スマートフィルター版 (`dm-checker_SmartFilter`)**
   - **対象**: 高校生〜大人（青年以上）
   - **特徴**: 大人向けの最新詐欺パターン16項目チェックに加え、学生にチェックで学生用の項目が増えます
   - 
2. **🧒 ファミリー版 (`dm_checker_family`)**
   - **対象**: 小中学生〜保護者
   - **特徴**: スマートフィルター版に加え「小中学生モード」が追加されたもの。切り替えて使えるようにしてあります。難しい言葉を避け、AIが「なかよくしていいか」を3つのポイントでやさしく教えるようにプロンプトを設定してあります

## 機能

### 🔍 アカウント・DM分析
- 公開アカウントはワンクリックでGrok自動分析
- 鍵アカウントはツイートをコピペして分析
- ハンドルをインライン編集して任意のアカウントを調査

### 🚩 チェックリスト（20項目）
- 投資・副業・ロマンス・海外案件・グルーミング・なりすましなど詐欺パターン網羅
- 若年層向け追加チェック項目（オーディション名目・秘密の撮影など）
- カテゴリ別ON/OFFで自分の用途に絞って表示

### ⚖️ 通常版のフィルター強度
- 弱・普通・強の3段階でGrokの判定指示を調整<br>
（小中学生モードは誤操作防止のため固定です）
### 🛡️ 判定システム（内部ロジック）
AIに丸投げせず、ブラウザ側で最新詐欺手口をチェックする「ダブル判定」機能

- **16種の不審シグナル**: 金銭要求・外部誘導・なりすまし等、コード内のリストに基づきリアルタイム検知
- **ローカル・スコアリング**: 各項目に重み付け（+1〜+3）を行い、5段階のリスクレベルを独自に算出
- **即遮断（BLOCK）ルール**: 「送金要求×外部誘導」など、危険な組み合わせを検知した瞬間に警告
- **いじめ・グルーミング対策**: 若年層向けには「二人の秘密」や「暴言」を厳重警戒
🐾 ローカル判定結果をGrokに共有し、AIが「怪しい点」を深掘りして判定してくれます

### 📋 信頼リスト
- アカウントごとに「信頼／要注意／危険」を登録・管理
- 登録情報はGrokクエリに自動反映

### ⚙️ プロンプトカスタマイズ
- 詐欺チェックの冒頭・末尾・ツイート分析プロンプトを自由に編集
- デフォルトへのリセット機能付き

### 🧒 キッズモード（ファミリー版のみ）
- 小学校高学年〜中学生向けのやさしい判定文に切り替え
- 子供向けチェックリストに自動切り替え
- 見守りガイド・安全ヒント表示

### 🔒 プライバシー
- 入力内容は外部サーバーに送信されません
- データはブラウザローカルに保存

## 注意事項

- **デベロッパーモード専用**です（Chrome Web Store未公開）
- 判定結果はあくまで参考です。最終判断はご自身でお願いします
- DM相手には判定結果は見えません（秘密にゃ🐾）
- Xアカウントが必要です
- テストしたところGrokは現在活発じゃないアカウントはだいたい怪しいと騒ぎだします
- 精度よりAI（他の人）に繋げることが重要だと思って自由に騒がせています
